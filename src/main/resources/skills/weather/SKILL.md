# Weather Skill

## Purpose
This skill allows the agent to check the current weather for a specified location.

## Parameters
- `location` (String): The city or area to get the weather for.

## Implementation
```java
// Mock implementation for demonstration
return "The weather in " + location + " is sunny with a high of 25°C.";
```
